# AI-Based-Blind-Spot-Detection-Systems
BlindSpot AI is a real-time driver assistance system that detects vehicles, pedestrians, and road hazards using a single camera. It combines YOLOv8 object detection with MiDaS depth estimation to calculate distances and trigger audio-visual alerts in danger zones. Built with Python and OpenCV, it runs on CPU, offering a low-cost ADAS solution.
